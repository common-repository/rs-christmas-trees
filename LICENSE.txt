== Description ==

Add nice looking animation effect of falling snow to your Wordpress site and enjoy winter and Christmas.

Simulate snow storm by adjusting the snow falling speed.

This snow falling WP plugin uses jSnow JQuery plugin and no images.

christmas is an easy-to-use Christmas plugin that can add snow, top and bottom Christmas banner to your WordPress website. All the effects are togglable so you can choose the ones you want.

 
Christmas plugin feature:

    1.Decorate your header and footer by gorgeous images
    2.Decorate your website by using snow fall
   

== Installation ==

1. Upload `rabbitseo.zip` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the Rs Christmas Trees menu item to configure the plugin.
